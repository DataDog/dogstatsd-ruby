# CHANGELOG

2.0.0/ UNRELEASED
=================

### Breaking changes
#### Namespace

The `Statsd` is now namespaced under the Datadog module.

To update:
- `require 'statsd'` -> `require 'datadog/statsd`
- `Statsd` -> `Datadog::Statsd`

### Notice
